QCD Example Codes

Below you can find fortran routines, card files and scripts in order
to do a standard QCD analysis of event shape distributions and
inclusive charged particle distributions on ALEPH data and full
MC (LEP1), as well as with some Monte Carlo generator.

Note : the analysis is histogram and not Ntuple based.
 
 

  1. Analysis of ALEPH data or full MC
  2. same analysis with PYTHIA

 

 
  1. Analysis of ALEPH data or full MC:
  • source code for ALPHA analysis :   qcd_analysis.F
  • definition of histogram identifiers :  histonaming.h
  • package of utility routines :                qcd_utilities.F
  • cards file for ALPHA analysis :        qcd_analysis.cards
  • example FILI cards for LEP1  :
                • da94_mini.filicards
                • da94_mini.sruncards
                • mc94_qqbar_mini.filicards

  • option file for ALPHARUN :             qcd_analysis.optx
      • To be used as alpharun -F qcd_analysis.optx

  • alternatively use this script :             qcd_job.submit
  • together with this makefile :             qcd_analysis.mk
      • To be used as bsub < qcd_job.submit
